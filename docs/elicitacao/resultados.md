# Resultado

## 1. Objetivo

<p align="justify">
O objetivo desse documento é reunir todos os requisitos levantados pelo grupo durante a fase de elicitação de requisitos., utilizando das técnicas: brainstorm, instrospecção, observação e questionário.
</p>
